# Blackboard Style Learning Platform

A web-based learning platform prototype created during a hackathon.

## Project Overview
This project was developed as part of a team hackathon and is inspired by classroom management and online learning platforms similar to Blackboard.

The platform focuses on creating a simple and organized interface for online learning activities, classroom interaction, and user-friendly navigation.

## Features
- Classroom style interface
- Web-based frontend design
- Multiple webpage structure
- Interactive UI elements
- Organized learning layout

## Technologies Used
- HTML
- CSS
- JavaScript
